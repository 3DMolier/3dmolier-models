# GitHub clean re-upload instructions

Use GitHub Desktop.

1. Open the local folder of your `3dmolier-models` repository.
2. Delete every old file and folder inside it.
3. Do NOT delete the hidden `.git` folder.
4. Unzip this archive.
5. Copy all files and folders from the unzipped archive into the repository root.
6. In GitHub Desktop, write Summary: `Clean rebuild with TurboSquid first images`
7. Click `Commit to main`.
8. Click `Push origin`.
9. Wait a few minutes.
10. Open https://3dmolier.github.io/3dmolier-models/ in Incognito or press Ctrl+F5.

This project uses direct TurboSquid JPG preview URLs for product cards.
