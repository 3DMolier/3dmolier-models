Как полностью заменить проект на GitHub:

1. Откройте репозиторий 3dmolier-models на GitHub.
2. Удалите старые файлы проекта. Через браузер это можно сделать по одному файлу: откройте файл -> Delete this file -> Commit changes.
3. Обязательно удалите старую папку models и старую папку assets, чтобы не остались старые локальные изображения и битые пути.
4. Самый удобный способ: GitHub Desktop. Clone repository -> удалить все файлы локально -> распаковать этот архив в папку репозитория -> Commit -> Push.
5. Если через браузер: распакуйте архив, загрузите файлы и папки частями меньше 100 файлов за раз. Не загружайте ZIP.
6. После обновления GitHub Pages откройте сайт и нажмите Ctrl+F5.
